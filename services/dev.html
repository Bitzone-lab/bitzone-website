DEV

Desarrollo de software

Significa para nosotros crear aplicaciones tecnológicas utilizando herramientas de programación, técnicas y patrones con el fin de brindar una solución a uno o muchos problemas del entorno. Además, involucra realizar tareas creativas y con ingenio en el plazo establecido conforme a lo que se requiere.

Creamos soluciones

Entre las actividades que realizamos para brindar las mejores soluciones están el análisis de la lógica de negocio, análisis de la BD, revisión de código, desarrollo de APIs y pruebas unitarias así como las de pruebas de integración.

También encontramos otras actividades como las de documentación para dar soporte al equipo y registrar las mejores prácticas.

Nos adaptamos a ti

Nos enfocamos en crear interfaces web que se ajusten a los dispositivos de nuestros clientes con el fin de mejorar la velocidad de carga así como mantener el contenido en cada una de ellas.

Nuestras tecnologías

Dependiendo de la complejidad del proyecto las tecnologías son variables, por ello, tratamos siempre de adaptarnos a las necesidades de nuestros clientes. Estas son algunas de las tecnologías que hemos venido utilizando en nuestros proyectos:

  • Flask, React y NextJS
  • Python, Javascript y Typescript
  • Lucidchart, Instagantt, Coda y Jira
  • Docker y Docker Compose
  • Git. Github y Gitlab
